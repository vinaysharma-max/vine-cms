---
title: Authors
description: Retrieve author information and profiles from your Vine workspace
---

The Authors API returns all authors in the workspace attached to the API key. Use it to render bylines, author cards, contributor directories, or editorial profile pages.

## Endpoint

```text
GET /api/public/v1/{API_KEY}/authors
```

### Query Parameters

Optional pagination is available:

- `limit` (number): positive integer page size, up to `100`
- `cursor` (string): opaque cursor returned by the previous response

## Example Request

```bash
curl "https://vinecms.tech/api/public/v1/{API_KEY}/authors?limit=20"
```

## Example Response

```json
{
  "ok": true,
  "authors": [
    {
      "id": "author_id",
      "name": "Ava Stone",
      "email": "ava@example.com",
      "about": "Writes product and editorial content.",
      "socialLinks": {
        "x": "https://x.com/ava"
      }
    }
  ],
  "pagination": {
    "limit": 20,
    "nextCursor": null,
    "hasMore": false
  }
}
```

## Response Fields

Each author object contains:

- `id` (string): internal author identifier
- `name` (string): display name
- `email` (string): author email
- `about` (string): author bio or description
- `socialLinks` (object): keyed social profile links
- `pagination` (object): collection pagination metadata

## Use Cases

- author biography sections
- contributor directory pages
- author selectors in external apps
- byline enrichment on post pages

## Example Usage

```js
const response = await fetch(
  'https://vinecms.tech/api/public/v1/{API_KEY}/authors?limit=20'
);
const data = await response.json();

const primaryAuthor = data.authors[0];
console.log(primaryAuthor.name);
console.log(primaryAuthor.socialLinks);
console.log(data.pagination.nextCursor);
```
