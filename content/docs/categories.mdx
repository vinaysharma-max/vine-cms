---
title: Categories
description: Retrieve all available categories from your Vine workspace
---

The Categories API returns all categories for the workspace attached to the API key. Use it to build navigation, section pages, content groupings, and CMS-driven filtering UIs.

## Endpoint

```text
GET /api/public/v1/{API_KEY}/categories
```

### Query Parameters

Optional pagination is available:

- `page` (number): positive integer page number
- `pageSize` (number): positive integer page size, up to `100`

If you omit both pagination parameters, Vine returns the full collection.

## Example Request

```bash
curl "https://vinecms.tech/api/public/v1/{API_KEY}/categories?page=1&pageSize=20"
```

## Example Response

```json
{
  "ok": true,
  "categories": [
    {
      "id": "category_id",
      "name": "Product",
      "slug": "product"
    }
  ],
  "pagination": {
    "page": 1,
    "pageSize": 20,
    "totalItems": 1,
    "totalPages": 1,
    "hasNextPage": false,
    "hasPreviousPage": false
  }
}
```

## Response Fields

Each category object contains:

- `id` (string): internal category identifier
- `name` (string): display name
- `slug` (string): URL-friendly category slug
- `pagination` (object): collection pagination metadata

## Use Cases

- category navigation
- section landing pages
- taxonomy selectors
- content grouping in external apps

## Example Usage

```js
const response = await fetch(
  'https://vinecms.tech/api/public/v1/{API_KEY}/categories?page=1&pageSize=20'
);
const data = await response.json();

data.categories.forEach((category) => {
  console.log(category.slug, category.name);
});

console.log(data.pagination.hasNextPage);
```
