title: Tags
description: Retrieve all available tags from your Vine workspace
---

The Tags API returns all tags for the workspace attached to the API key. Use it to build tag clouds, topic navigation, search facets, or external taxonomy tooling.

## Endpoint

```text
GET https://vinecms.tech/api/public/v1/{API_KEY}/tags
```

### Query Parameters

Optional pagination is available:

- `limit` (number): positive integer page size, up to `100`
- `cursor` (string): opaque cursor returned by the previous response

## Example Request

```bash
curl "https://vinecms.tech/api/public/v1/{API_KEY}/tags?limit=20"
```

## Example Response

```json
{
  "ok": true,
  "tags": [
    {
      "id": "tag_id",
      "name": "Updates",
      "slug": "updates"
    }
  ],
  "pagination": {
    "limit": 20,
    "nextCursor": null,
    "hasMore": false
  }
}
```

## Response Fields

Each tag object contains:

- `id` (string): internal tag identifier
- `name` (string): display name
- `slug` (string): URL-friendly tag slug
- `pagination` (object): collection pagination metadata

## Use Cases

- **Tag Clouds**: Display all tags with post counts
- **Filter Menus**: Build dropdown or checkbox filters for posts
- **Navigation**: Create tag-based navigation links
- **Search**: Use tags to enhance search functionality

## Example Usage

```js
const response = await fetch(
  'https://vinecms.tech/api/public/v1/{API_KEY}/tags?limit=20'
);
const data = await response.json();

data.tags.forEach((tag) => {
  console.log(`${tag.name} (${tag.slug})`);
});

console.log(data.pagination.limit);
```
